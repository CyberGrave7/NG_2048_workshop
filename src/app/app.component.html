Angular 2048

HOW TO PLAY:

Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

ABOUT:

his simple implementation of the 2048 game is intended to serve as an example for an Angular workshop that took place on 26th of March 2017. The task for this workshop was requested by students from the second HackBulgaria Angular course that was organized by me (Ilia Idakiev). Source code for this solution can be found on GitHub and a dynamic version can be found here.
